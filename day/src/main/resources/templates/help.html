Installation and use of TronLink wallet plugin

1、First install the TronLink wallet plugin

Install

2、If you don't have a TronLink account, sign up for a TronLink account

Chrome User Guide

3、Login to tronlink wallet

Steps of mining

1、You need to have a certain amount of TRX, mainly to pay for contract calls

2、You need to have a certain amount of DAY, which is mainly used to pay for the creation of NFT and collateral mining

3、Click Mint

4、You can choose either DAY POOL or DAY + NFT POOL in Service; DAY POOL supports mining using only DAY, DAY + NFT POOL supports mining using DAY + Magic Box NFT. The annualized return of the two mining pools will have a difference of 5 times, the mining pool using NFT mining has a higher rate of return.

5、Click Stake, enter the number of DAYs pledged and you're ready to mine (if it's NFT mining you also need to select an NFT)! It's that simple.

How to buy magic box

1、Just go to the NFT mining page and then click the Buy Box button

2、Or click the Buy And Open Box button. This will mean buying and opening the box directly.

3、A fee of 100 MH Tokens is required to buy an Box.Please make sure you have enough MH Tokens in your account.

How to Create NFT

1、Just click My-NFT go to the My NFT page

2、Upload the image you need to create, fill in the title, and click Create NFT.

3、A fee of 100 DAY Tokens is required to create an NFT.Please make sure you have enough DAY Tokens in your account.

How to trade NFT.

1、Just click My-NFT go to the My NFT page

2、You can add NFT to the one-bite market, or you can add NFT to the auction market, or you can transfer directly to a particular user.

3、You can view the NFTs you have added to the market here.

4、Now You can click Market go to the Market page. Here you can buy directly or bid at auction.
