  • NFT Mining
    The Great NFT
    MH token is all mint out through DAY and NFT magic hamper cards. Genesis mining is done using DAY. Combined mining(DAY + NFT) will be opened after the end of the Genesis mining.
  • NFT Management
    The Great NFT
    The current management of NFT mainly includes: NFT production, NFT trading, and NFT auction. Users can perform various transactions and manipulations of NFT on the platform.

    The Great NFT

Token

DAY is the platform token circulating on the platform. In the future, after the main network of the platform is launched, it will serve as the circulating token of the main network. The governance of the main network needs to be carried out through DAY.

  • MH creation mining only supports DAY, MH tokens are Magic Hamper tokens, and Magic Hamper purchases only support MH tokens.
  • DAY is the designated token for NFT production, and the platform supports the majority of users to make NFTs, and trade and auction works.
