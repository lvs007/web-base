Installation and use of TronLink wallet plugin

1、First install the TronLink wallet plugin

2、If you don't have a TronLink account, sign up for a TronLink account

3、Login to tronlink wallet

4、Switching networks: switch to TRON Nile test network

Steps of mining

1、Need to apply for TRX of test network, mainly to pay for contract calls

2、You need to apply for a test network DAY, which is mainly used to pay for the creation of NFT and collateral mining

3、Select mint in Service

4、You can choose either DAY POOL or DAY + NFT POOL in Service; DAY POOL supports mining using only DAY, DAY + NFT POOL supports mining using DAY + Magic Box NFT. The annualized return of the two mining pools will have a difference of 5 times, the mining pool using NFT mining has a higher rate of return.

5、Click Stake, enter the number of DAYs pledged and you're ready to mine (if it's NFT mining you also need to select an NFT)! It's that simple.

How to buy magic box

1、Just go to the NFT mining page and then click the Buy Magic Box button. It's that easy!
